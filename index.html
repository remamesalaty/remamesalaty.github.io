Web Design & UX

Website & UX Design for a Family Café

My Role: UX/UI Design, Visual Design, Frontend Development

Overview

A website designed for a family café in Vienna, combining a welcoming atmosphere with clear user navigation.

Problem

The café only had an Instagram presence, making it difficult for parents to find structured information or make reservations.

Goal

Design a website that clearly communicates the concept and makes it easy for parents to explore and book.

The Process

01 Research: Understanding parents as target users
02 Structure: Planning pages and navigation
03 Design: Creating a friendly and trustworthy visual identity
04 Development: Building responsive layouts with HTML & CSS

Key Design Decisions

  • Safe & Friendly: Soft colors and rounded typography to create a safe, family-friendly feeling.
  • Clear Navigation: Designed to reduce confusion for busy parents.
  • Mobile-First: Ensuring easy access on phones for on-the-go parents.
  • Visual Trust: Using images and video storytelling to build authentic trust.

Outcome

The final design improves clarity, enhances user trust, and supports key actions like reservations.

Future Improvements

If expanded further, this project could include features such as online booking systems, user accounts, and performance dashboards to support business growth.
